Abstract:
Generating high quality summary that are succinct, syntactic correct, and accurate is an on-going challenge. Previously, there have been attempts to solve the repetition issue of Recursive Neural Network with attention and coverage. However, maintaining the original focus remains a challenge.

Given plenty of redundant news, it can be inferred that known similar source documents should also produce similar abstracts. If the semantic meaning is always reduced by the same magnitude from source to summary under the same summarization model, then I propose to use these similarity relationships to predict summaries that deviate from their parents.

Secondly, we could deploy an ensemble of summarization models; the summary most “neutral” in semantic and agreed upon by most models is most likely to be true. The final app would be a lightweight TensorFire model deployed to user’s device, allowing private and fast summarization.

def filter_docs(corpus, texts, labels, condition_on_doc):
    """
    Filter corpus, texts and labels given the function condition_on_doc which takes
    a doc.
    The document doc is kept if condition_on_doc(doc) is true.
    """
    number_of_docs = len(corpus)

    if texts is not None:
        texts = [text for (text, doc) in zip(texts, corpus)
                 if condition_on_doc(doc)]

    labels = [i for (i, doc) in zip(labels, corpus) if condition_on_doc(doc)]
    corpus = [doc for doc in corpus if condition_on_doc(doc)]

    print("{} docs removed".format(number_of_docs - len(corpus)))

    return (corpus, texts, labels)

def document_vector(word2vec_model, doc):
    # remove out-of-vocabulary words
    doc = [word for word in doc if word in word2vec_model.vocab]
    return np.mean(word2vec_model[doc], axis=0)

def most_similar(i, X_sims, topn=None):
    """return the indices of the topn most similar documents with document i
    given the similarity matrix X_sims"""

    r = np.argsort(X_sims[i])[::-1]
    if r is None:
        return r
    else:
        return r[:topn]

Evaluation of translation and summerization model have typically been evaluated using BLEU score, ROUGE-N. ROUGE-N metric ( ), is the N-gram measure between the model and the gold summary. The value offered by my algorithm is posing additional constraint on the test summary where there is no golden summary available. Further, it leverages the benefit of assemble, and let different summerization output and/or model to constraint each other from semantic deviation in each unique case.
